
# How to build

This is built using CMake. You will also need the required libraries to build. 
This project is tested on Arch Linux. This _should_ also work on Windows, given
that it runs fine in Wine. 

```sh
mkdir build
cd build
cmake ..
make
```

# Required libraries
- GLEW
- GLFW
- OpenGL
- FreeType
- AssImp
- JsonCpp
